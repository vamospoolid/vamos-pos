===========================================================
VAMOS POOL & CAFE - PANDUAN INSTALASI KASIR (V2 - SMART HARDWARE)
===========================================================

APA YANG BARU DI V2:
1. Smart Hardware Detection: Aplikasi otomatis mencari port Relay (CH340).
2. Auto-Fallback: Jika port manual salah, aplikasi akan mencari port yang benar.
3. Enhanced Logging: Log yang lebih detail untuk diagnosa hardware.

LANGKAH 1: INSTALASI DATABASE (Jika Belum Ada)
1. Install PostgreSQL 16 di laptop kasir.
2. Gunakan password default: "vamos-root" (atau sesuaikan dengan .env).
3. Buat database kosong bernama: "vamos_pos".

LANGKAH 2: INSTALASI APLIKASI
1. Jalankan Vamos_POS_Setup_x.x.x.exe atau langsung gunakan Vamos_POS_Portable.exe.
2. Shortcut akan muncul di Desktop (jika menggunakan Setup).

LANGKAH 3: KONFIGURASI HARDWARE (PENTING)
1. Gunakan file ".env" yang disertakan.
2. Secara default RELAY_COM_PORT=AUTO.
3. Jika ingin mengunci port tertentu, ubah menjadi RELAY_COM_PORT=COM3 (sesuaikan).
4. Letakkan file ".env" di folder yang sama dengan Vamos_POS_Portable.exe.

LANGKAH 4: SETTING PRINTER
1. Share printer dengan nama: RP58 (atau sesuaikan di Settings aplikasi).
2. Pastikan "File and Printer Sharing" aktif di Windows.

LANGKAH 5: JALANKAN APLIKASI
1. Buka VAMOS POOL POS.
2. Cek pojok kanan atas: Indikator Hardware harus berwarna HIJAU.
3. Jika tetap MERAH, cek kabel USB Relay dan pastikan driver CH340 terinstall.

TIPS: Jika port hardware berubah, aplikasi akan otomatis mendeteksi ulang saat restart.
